# AirWiki branding

`airwiki-mark.png` is the canonical AirWiki mark. The remaining files are
platform-specific derivatives of that source:

- `airwiki-app-icon.png`: cropped, normalized source for desktop derivatives.
- `github-avatar.png`: square organization avatar.
- `github-social-preview.png`: repository social preview.
- `airwiki.icns`: multi-resolution macOS application and DMG icon.
- `airwiki.ico`: multi-resolution Windows application and installer icon.
- `airwiki-window.rgba`: embedded 128 px application-window icon.
- `airwiki-tray.rgba`: embedded 24 px transparent tray icon.

The branding files are distributed under the repository's Apache-2.0 license.
Keep the source mark unchanged when regenerating derivatives, and verify small
sizes visually before replacing packaged assets.
